What is Dicoding?

Dicoding is an Indonesian-based online learning platform that focuses on tech education, particularly in programming and software development. It offers a variety of courses, certifications, and learning paths in fields like mobile app development, web development, cloud computing, machine learning, and more. Dicoding collaborates with leading technology companies such as Google, Microsoft, IBM, and AWS to create industry-relevant curricula.

Current Topic of Study

Basic Web Programming

Websites in today's era have become a primary necessity that cannot be ignored. Every sector, whether in business or education, can utilize websites as tools for promotion, information exchange, and more. According to data from World Wide Web Technology Surveys, out of all active websites, 88.2% use HTML and 95.6% use CSS. This course thoroughly covers HTML and CSS as the foundation of website development.

learning Path

In the Basic Web Programming material, I studied the basic material of HTML and CSS.

HTML

Learning HTML (HyperText Markup Language) teaches how to create the basic structure of a web page, including paragraphs, headings, images, and links. Students will learn to use HTML tags to build well-structured web content.

CSS

Learning CSS (Cascading Style Sheets) focuses on how to style and layout web pages. It allows students to control the appearance of HTML elements, such as colors, fonts, spacing, and positioning.

Flexbox

Learning responsive layout with Flexbox teaches how to create flexible and adaptive web designs that work seamlessly across various screen sizes and devices.
